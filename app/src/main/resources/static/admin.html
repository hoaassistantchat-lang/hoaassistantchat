Widget Partners

Generate API keys for partner websites that embed your HOA chat widget

How it works

1. Create an API key — Give it a name (e.g. "Sunset Hills Website") and list the domains the key is allowed to be used from. Only requests from those domains will be accepted.
2. Share the embed snippet — Copy the generated <script> tag and send it to your partner. They paste it before the closing </body> tag.
3. Revoke any time — Click Deactivate to instantly disable a key without affecting other partners.
